Jimac How to

Jimacとは？

Jimacは、歌枠や歌ってみたの字幕オーバーレイ素材をお手軽に作れるWebツールです。
カラーやフォントの変更、縁取りなど多彩な装飾機能を備え、レパートリーの管理や検索なども行えます。

YouTubeで#Jimacを検索
Twitterで#Jimacを検索

引用: グッバイ宣言 / FloweR

配信ツールの設定 (OBS)

ウィンドウキャプチャでJimacのウィンドウを取り込みます。
フィルタ機能より字幕表示部分以外をクロップ（トリミング）し、字幕の背景色をクロマキー/カラーキーで透過します。

歌詞ボード
字幕を入力またはコピペで登録することができます。（無料版は70行まで）

引用: シャルル / バルーン

字幕の切り替え

[] カウンター[表示行]を0に戻します。
[] (または左矢印キー) カウンターを1つ前に戻します。
[] (または右矢印キー) カウンターを1つ先に進めます。

背景色、文字色

字幕の背景色や文字色をカラーピッカーで変更できます。
RGB/HSL/HEXの数値での指定やスポイトでの色抽出も可能です。

フォント

字幕のフォントやフォントサイズを変更できます。
フォントサイズは楽曲ごとに保持されます。
（フォントの種類によって縁取りにトゲのようなノイズが出る場合があります）

【利用可能フォント】
Kiwi Maru / Yusei Magic / Hachi Maru Pop / Kaisei Opti

有料版では以下の11フォントが追加されます
Dela Gothic One / Klee One / Kosugi Maru / New Tegomin / Potta One / Shippori Mincho B1 / Yomogi / DotGothic16 / Rampart One / RocknRoll One / Stick

文字装飾と演出

字幕切り替え時のアニメーションを設定したり、テキストに縁取りを施すことができます。

Hint*
- フォントによっては縁取りにトゲついたノイズが入る場合があるため、気になる場合は縁取りをオフにしてください。
- アニメーション時の半透明部の色が綺麗に処理できない場合は、背景色を黒または白にすると違和感が少なくなります。

次に表示される歌詞を確認できます。

引用: ヨワネハキ / MAISONdes

楽曲検索（フィルタ）

楽曲のタイトル（歌詞の一行目）と登録されたタグから楽曲検索[フィルタ]を行えます。
複数の単語で検索したい場合はスペースで区切ります。（AND検索になります）
単語の先頭に-[マイナス記号]をつけることで除外検索を行えます。

楽曲一覧

各楽曲の歌詞の先頭がタイトルとなり、一覧表示されます。
6曲目以降はスクロールで表示されます。（無料版は5曲まで）

楽曲管理

[] 楽曲を追加します。9曲目以降は有料版へのアクティベートが必要です。
[] [] 選択中の楽曲を1つ上または下の(表示中の)楽曲と位置を入れ替えます。
[] 選択中の楽曲を削除します。
[] 楽曲一覧（フィルタ済みの内容）をクリップボードにコピーできます。

動画プレーヤー

YouTubeまたはニコニコ動画のURLを張り付けると動画プレーヤーが表示され、動画を再生できます。
練習用のカラオケ動画や、二次利用可能な音源などを登録してお使いください。

引用: シャルル／flower

タグ

楽曲検索 [フィルタ]に使えるタグ/キーワードを設定します。先頭が#である必要はありません。（無料版は20文字まで）

使い方のヒント
- [〇〇歌枠] などを設定しておくことで、歌枠のレパートリーを作ることができます。
- [バラード] など楽曲のジャンルを設定しておくことで、縛り歌枠などで使えるレパートリーが出せます。
- [練習中] と設定しておくことで、練習中の曲を除外検索（先頭に-をつけて検索）できるようになります。

データの取り扱いとバックアップ

よくある質問

Jimacは商用可能ですか？

Jimacは個人/法人、商用/非商用に関わらず誰でもお使いいただけます。
商用利用に際して有料版購入を行う必要もありません。

Jimacの利用にあたって製作者への連絡やクレジット表記は必要ですか？

利用の際の連絡や製作者のクレジット表記は不要です。
動画や配信の概要欄に十分なスペースがある場合に限り、当サイトへのリンク表記を頂ければ幸いです。
（Twitterでの告知や切り抜き動画などでは不要です）

また使い方ページにて #Jimac のYouTubeやTwitter検索リンクを設置しておりますので、 #Jimac のハッシュタグを付けていただきますと助かります。

Jimacのデータはサーバーに保存されますか？

Jimacに登録した歌詞データは、利用ユーザーのブラウザ記憶領域（ローカルストレージ）に保存されます。
Jimacは利用ユーザーが著作権を保持していない歌詞データ等を扱うことを想定しているため、全てのデータはサーバーに保存されません。 これにより、利用ユーザーは個人利用の範囲内で歌詞データを管理することができます。

一方でサーバーで管理されない登録データは、ブラウザのキャッシュクリアやPC故障によってデータを失うリスクがあるため、使い方ページの最下部にある『バックアップ機能』を使い、手動でのバックアップを行ってください。
データ消失に関する補填やサポートなどは一切行っておりません。

Jimacを利用した歌詞情報の配信/映り込みは著作権上問題ありませんか？

Jimacは著作物のデータを配布せず、サービスとしてデータの保存/管理も行わないため、他者の著作物に関しては一切関知しません。
歌詞データ等の他者著作物を利用する際は、著作者の許諾の範囲内または著作権法上許可される利用の範囲内で適切にお使いください。

歌詞データ以外の登録、字幕として以外の利用は可能ですか

利用規約から逸脱しない範囲の利用であれば問題ありません。

動画URLを設定したのですが再生できません。YouTubeやニコニコ上では見れるのですが……

YouTubeやニコニコでは他Webページ埋め込みでの再生を禁止する設定があります。埋め込み禁止設定のされている動画は再生できません。

スマホ表示やウィンドウを小さくすると見た目が壊れます

仕様です。
配信用ツールという性質上、ある程度長い横幅で使うことを想定したWebページですので、ウィンドウサイズが小さい場合の表示はあまり調整できていません。

壊れて動かなくなりました

保存されているデータが破損した可能性があります。
『使い方』ページの一番下にある『データのバックアップ』を行った後、『データの初期化』をしてみてください。

Jimacのバックアップデータを他者に配布することは可能ですか？

Jimacとしてバックアップデータの配布/共有を禁止はしませんが、他者著作物を含む場合は公衆送信等の権利侵害に当たる利用を行わないよう、十分にご留意ください。

意見要望や不具合の連絡などは送れますか

開発・運営者の小栗さえまで、メンションまたはDMでご相談ください。

Jimacの有料化

Jimac有料版は BOOTH にて1,000円で購入いただけます。購入後はJimacのサービス終了まで無期限で利用できます。
楽曲一覧部右下にある [] マークの入ったフォームに購入したコードを入力してアクティベートしてください。

有料版の機能

有料版にアクティベートされると以下の機能が解放されます。

  • 楽曲を9曲以上追加/管理可能になります。
  • 追加のフォント11種を利用できるようになります。
  • 楽曲一覧下部のバナー広告が無くなります。
  • 歌詞を70行以上登録可能になります。
  • タグを合計20文字以上登録可能になります。

プライバシーポリシー

Jimacに登録されたデータについて

Jimacに登録された歌詞データはブラウザのローカルストレージに保存され、サーバーに送信されません。
ブラウザのキャッシュクリアやPCの買い替えなどによりデータを失うと復元できなくなるため、データのバックアップを行いたい場合はサービス下部『使い方』にある、『データのバックアップ』の項にて手動でバックアップを行ってください。

Cookie

本サービスではログイン状態の確認、ユーザー利便性の向上、トラフィックデータ収集、広告配信のためにCookieを利用しています。

Cookieを利用する外部サービスは以下の2つです。
- Google Analytics
- Google AdSense

ユーザーはブラウザの設定によりCookieの送信を拒否することができます。
Jimacから収集した情報の GOOGLE による使用については こちら をご覧ください。

Google Analytics

Jimacではサイトの改善を行なうため、Google Analyticsを利用しています。

ユーザーはディスプレイ広告に対応する Googleアナリティクス機能によるデータ収集を無効化 することや、 広告設定 を使用してGoogleディスプレイネットワークで表示される広告をカスタマイズすることができます。

Google AdSense

※ 現在広告は表示しておりません。
Google AdSenseではCookieを使用して、本サービスや他のサービスへの過去のアクセス情報に基づいてGoogleを含む第三者配信事業者が広告を配信します。

ユーザーは Ads Preferences Manager で、インタレスト ベースでの広告掲載に使用される DoubleClick Cookie を無効にできます。 aboutads.infoページ では、インタレストベースでの広告掲載に使用される第三者配信事業者の Cookie を無効にできます。
